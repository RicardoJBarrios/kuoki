environmentQueryConfigFactory

  • environmentQueryConfigFactory(config?: EnvironmentQueryConfig): Required<EnvironmentQueryConfig>
    • Defined in packages/environment/src/lib/query/environment-query-config-factory.function.ts:19

    Returns the environment query configuration with default values for empty parameters.

    The default EnvironmentQueryConfig value is { interpolation: ['{{', '}}'], transpileEnvironment: false }.

    example
    environmentQueryConfigFactory();
    // {interpolation:['{{','}}'],transpileEnvironment:false}
    environmentQueryConfigFactory({ interpolation: ['(', ')'] });
    // {interpolation:['(',')'],transpileEnvironment:false}
    environmentQueryConfigFactory({ transpileEnvironment: true });
    // {interpolation:['{{','}}'],transpileEnvironment:true}

    Parameters

    • Optional config: EnvironmentQueryConfig

      The partial environment query config.

    Returns Required<EnvironmentQueryConfig>

    The environment query configuration with all default values for empty parameters.
